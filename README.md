# ИДЕЯ: сайт с физическим анти-стресс симуляциями

## Цель проекта

Разработать web-приложение, в котором можно запускать различные физические симуляции, на которых можно долго залипнуть и поэкспериментировать с физическими явлениями.

## Описание проекта

Сайт с физическими анти-стресс симуляциями предназначен для отдыха и расслабления. На сайте можно смотреть на различные физические симуляции и взаимодействовать с ними.

## Требования к интерфейсу

- Главная страница содержит название проекта и кнопки выбора симуляций
- Симуляции 2D (примеры):  сплошная среда, задача трех тел, колебания струны, теплопроводность
- Симуляции должны содержать красивую визуализацию, кнопки для регулировки параметров (если такие предполагаются)

## Симуляции, которые мы планируем реализовать

- Задача трёх тел. Три точечные массы притягивают друг друга по закону всемирного тяготения. Начальные условия которые можно задавать телам - массы, скорости и координаты (всё что нужно для задачи Коши).
- Теплопроводность. Распространение тепла на плоскости с возможностью задавать начальное распределение температур, а также граничные условия - задача Дирихле (постоянное распределение температуры на границе).
- Сплошная среда. Симуляция уравнений Навье-Стокса в 2D. Граничные условия - задача Неймана. Возможность взаимодействия со средой посредством мышки. Визуализация - движения частиц (красителя) в динамически меняющемся поле скоростей в соответствии с уравнениями.
- Колебания струны. Одномерное уравнение колебаний. Оба конца струны закреплены, длина струны конечная. Возможность придать струне произвольную начальную форму.
